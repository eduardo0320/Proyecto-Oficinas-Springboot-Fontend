# Sistema de Control de Acceso a Oficinas - Frontend

Proyecto de demostración desarrollado como parte de un curso universitario. Al ser un proyecto académico, puede contener errores o áreas de mejora.

---

## Tecnologías utilizadas

- HTML5
- JavaScript (Vanilla)
- Bootstrap 5
- Bootstrap Icons
- Chart.js
- jsPDF
- SheetJS (xlsx)

---

## Requisitos

- Navegador web moderno
- Backend corriendo en `http://localhost:8080`

## Cómo correr el proyecto

1. Clonar el repositorio
2. Asegurarse de que el backend esté corriendo
3. Abrir `HTML/login.html` en el navegador

---

## Funcionalidades

- Login con JWT
- Gestión de personas y oficinas (CRUD)
- Control de ingresos y salidas
- Búsqueda y paginación
- Exportación a PDF y Excel
- Estadísticas con gráficos
- Notificaciones toast
- Control de acceso por roles (ADMIN, REGISTRADOR, VISOR)
